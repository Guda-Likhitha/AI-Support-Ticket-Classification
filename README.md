# 🎫 AI Support Ticket Classification System

An AI-powered Support Ticket Classification System built using Python and Django. This application helps automate customer support workflows by categorizing support tickets into predefined categories, reducing manual effort and improving response efficiency.

## 🚀 Features

- 📝 Create and manage support tickets
- 🤖 AI-based ticket classification
- 📊 Dashboard for ticket monitoring
- 🔍 Automatic categorization of customer issues
- 📂 Organized ticket management
- 🌐 User-friendly web interface
- 📈 Improved support team productivity

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Guda-Likhitha/AI-Support-Ticket-Classification.git
```

### Navigate to Project Directory

```bash
cd AI-Support-Ticket-Classification
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Mac/Linux:

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start the Development Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```

## 🎯 How It Works

1. User submits a support ticket.
2. The AI model analyzes the ticket description.
3. The system predicts the appropriate category.
4. Ticket is stored and displayed on the dashboard.
5. Support teams can prioritize and resolve issues efficiently.
